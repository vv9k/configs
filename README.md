# My dot files

## Info
The configuration files are rendered with [`mold`](https://github.com/vv9k/mold). The template files are located in the [`mold`](https://github.com/vv9k/configs/blob/master/mold) directory, together with the [`change_theme.sh`](https://github.com/vv9k/configs/blob/master/mold/change_theme.sh) script that is responsible for changing the themes. It uses values in [`context.yml`](https://github.com/vv9k/configs/blob/master/mold/context.yml) to render the files.
